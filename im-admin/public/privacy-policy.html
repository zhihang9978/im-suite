志航密信 隐私政策

更新日期：2025年10月11日 | 生效日期：2025年10月11日

重要提示：请您仔细阅读本隐私政策。一旦您使用或继续使用志航密信服务，即表示您同意我们按照本隐私政策收集、使用、存储和共享您的相关信息。

一、我们收集的信息

1.1 您主动提供的信息

信息类型 收集目的 是否必需
手机号码 账号注册、登录验证、找回密码 是
昵称、头像 个人资料展示 否
聊天内容 消息传输、内容审核 是
通讯录 查找好友（需用户授权） 否
文件、图片、视频 文件传输、存储 否

1.2 我们自动收集的信息

  • 设备信息：设备型号、操作系统、唯一设备标识符、IP地址
  • 日志信息：登录时间、访问时间、操作记录
  • 位置信息：IP地址推断的大致位置（不精确定位）
  • 性能数据：应用崩溃、网络状态（用于优化服务）

二、我们如何使用信息

我们仅在以下目的内使用您的信息：

  1. 提供服务：账号管理、消息传输、语音/视频通话
  2. 改进服务：分析用户行为、优化功能
  3. 安全保障：身份验证、防止欺诈、内容审核
  4. 法律合规：遵守法律法规、配合监管部门
  5. 通知服务：发送系统通知、服务更新

三、信息存储

3.1 存储位置

  • 服务器位置：中国大陆
  • 云存储：阿里云/腾讯云（符合等保要求）
  • 备份：异地备份（加密存储）

3.2 存储期限

信息类型 存储期限
账号信息 账号注销后48小时删除
聊天记录 用户删除后前台不可见，后台保留90天（合规要求）
日志信息 90天
文件（图片、视频） 用户删除后即刻删除

3.3 安全措施

  • HTTPS加密传输（TLS 1.2+）
  • 数据库敏感字段加密（AES-256）
  • 访问权限控制（RBAC）
  • 定期安全审计和漏洞扫描

四、信息共享与披露

4.1 不会共享的情况

我们不会出售、出租或交易您的个人信息给任何第三方。

4.2 可能共享的情况

  • 第三方服务商：云存储（阿里云OSS）、内容审核（阿里云）、短信服务
  • 关联公司：在合并、收购时可能转移信息（需通知用户）
  • 法律要求：应政府部门、司法机关要求提供信息
  • 用户同意：经您明确同意的其他情形

五、您的权利

您对个人信息享有以下权利：

5.1 访问权

您可以随时查看您的个人资料、聊天记录

操作路径：设置 → 个人资料

5.2 更正权

您可以修改昵称、头像、个人签名等信息

操作路径：设置 → 个人资料 → 编辑

5.3 删除权

您可以删除聊天记录、文件，或注销账号

操作路径：设置 → 账号管理 → 注销账号

5.4 导出权

您可以导出您的个人数据（JSON格式）

操作路径：设置 → 隐私 → 导出我的数据

5.5 撤回同意

您可以随时撤回对非必要权限的授权（如通讯录、位置）

操作路径：设置 → 隐私 → 权限管理
